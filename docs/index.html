The Problem & Solution

The Problem

A multi-agent pipeline needs a producer at the top of the funnel — something that turns a topic into a draft other agents can grade, vet, and ship. Without it, orchestration has nothing to orchestrate.

The Solution

Worker takes a topic on-chain and returns a structured, sourced research brief. It writes with an LLM when a key is present, and degrades to a deterministic offline draft otherwise — so it always delivers something gradeable.
